Installation:
Unzip to a directory <octave-dir> wich doesnt have space chars i.e. C:\Octave\octave3.6.4_gcc4.6.2\ keeping the directry structure in the archive
Copy octave3.6.4_gcc4.6.2.lnk to any convient loaction and edit its properites respectvely to point to <octave-dir>\bin\octave.exe and icon <octave-dir>\doc\octave\icons\octave-logo.ico
Note for windows 8 users: As a workaround for a gnulib windows 8 compatibility bug, keep the command line switches ' -i --line-editing' in the octave.exe shortcut (i.e. <octave-dir>\bin\octave.exe -i --line-editing)
Copy octave3.6.4_gcc4.6.2_docs.lnk to any convient loaction and edit its properites respectvely to point to <octave-dir>\doc\octave and icon <octave-dir>\doc\octave\icons\octave-logo.ico
This archive doesn't contain any editor !
In order to install Notepad++ (recopmmended) as an editor:
    a. download the installation package from http://notepad-plus-plus.org/
    b. install, edit <your octave dir>\share\octave\site\m\startup\octaverc
    c. uncomment the line which set octave default editor: EDITOR('C:\\Program Files\\Notepad++\\notepad++.exe');
