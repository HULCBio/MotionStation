Multiple Access

The Multiple Access category contains a SIMULINK Multiple Access Library. This library includes analog time division multiple access (TDMA), digital TDMA function blocks, frequency division multiple access (FDMA), and code division multiple access (CDMA) examples. There are no MATLAB functions directly designed with multiple access. Use the functions provided in the Modulation and Demodulation category and the filters provided in the Signal Processing Toolbox.

You can view a list of all SIMULINK blocks in this category.
