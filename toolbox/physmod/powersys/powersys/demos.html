SimPowerSystems Demos

SimPowerSystems models and simulates electrical power systems and drives within the Simulink environment. Applications of this product include analysis and modeling of power utility distribution networks and self-contained power systems such as those for ships, aircraft, and spacecraft.

The product covers the following areas:

  • Power system networks
  • Electric machinery
  • Power electronics
  • Control and measurement
  • Three-phase machines

Simple Models

Linear Filter: Analysis of steady-state operation of a linear electrical circuit

Transient Analysis: Transient analysis of a linear circuit

Linear Transformer: Three-winding distribution transformer

Three-phase Saturable Transformer: Energization of a three-phase saturable transformer

Current Transformer Saturation: Measurement distortion due to saturation of a current transformer

AC Surge Arrester: Surge arresters used in a series and shunt compensated transmission

Single-phase Line: Time domain and frequency domain testing of a single phase line

Three-phase Line: Single-phase energization of a three-phase line

Three-phase Dynamic Load and Programmable Source: Use of the Three-Phase Dynamic Load and Three-Phase Programmable Voltage Source blocks

Saturable Transformer with Hysteresis: Simulation of hysteresis in a Saturable Transformer block

Three-phase Static Var Compensator: Steady-state and dynamic performance of a static var compensator (phasor)

Cassie and Mayr Arc in Circuit Breaker Interruption: Application of, and difference between, the Cassie and Mayr arc models for simulating circuit breaker interruption

Variable Inductance:  Modeling a variable Inductor block

Power Electronics Models

Single-phase Rectifiers: Two identical single-phase rectifiers (discrete)

Three-phase Rectifier: Diode block simulating a three-phase rectifier and comparing continuous vs discrete simulations

Thyristor Converter: Thyristor model and universal bridge simulating a three-phase six-pulse converter and demonstrating harmonic analysis using the Powergui FFT tool

Ideal Switch: Inductive current chopping by an Ideal Switch block

MOSFET Converter: MOSFET in a zero-current quasi-resonant converter

GTO Buck Converter: GTO thyristor in the buck converter topology

DC/DC and DC/AC PWM Converters: Universal bridge and discrete PWM pulse generator with harmonic analysis of PWM waveforms using the Powergui FFT tool (discrete)

Three-phase Two-level PWM Converters: Three-phase two-level PWM voltage source converters (discrete)

AC/DC Three-level PWM Converter: AC/DC Three-level PWM converter (discrete)

Three-phase SV-PWM Converter: Open-loop speed control of an induction motor using a space-vector PWM modulator (discrete)

Three-phase Three-level PWM Converter: Three-phase three-level inverter and discrete three-phase PWM generator with harmonic analysis of PWM waveforms using the Powergui FFT tool (discrete)

Three-phase 48-pulse GTO Converter: Three-level converters and zigzag phase-shifting transformers in a 48-pulse square-wave GTO converter with harmonic analysis using the Powergui FFT tool

Universal Bridge in DC-AC PWM Converter: Example of AC-DC-AC converter illustrates use of Universal Bridge and Multimeter blocks as well as control blocks of the Extras library (discrete)

Machine Models

Synchronous Machine: Synchronous generator powered by hydraulic turbine with excitation and governor systems

Asynchronous Machine: Asynchronous machine in an open-loop speed control on a 3 HP, 220 V industrial motor

Wind-turbine Generator: Wind-turbine asynchronous generator in isolated network

Synchronous Machine and Regulator: Nonlinear control of a hydraulic turbine and a synchronous generator

Transient Stability of 2 Machines with PSS and SVC: Transient stability of a two-machine transmission system with Power System Stabilizers (PSS) and Static Var Compensator (SVC) (phasor)  (requires Control System Toolbox)

Performance of Three PSS for Interarea Oscillations: Comparison of three Power System Stabilizers (PSS) using Kundur's four-machine two-area test system (phasor)  (requires Control System Toolbox)

Drive Models

Chopper-fed DC Motor Drive: Discretization of continuous DC motor drive model for faster simulation (discrete)
