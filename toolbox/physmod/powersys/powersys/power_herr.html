Warning - Documentation Files Not Found

MATLAB cannot find the online documentation files used by the Help browser.

By default, the documentation files are automatically installed to the $matlabroot/help directory when MATLAB is installed. There might be a problem with the files, they might have been removed, or they might not have been installed.

If the documentation files are in the help directory, please report this problem at http://www.mathworks.com/support/forms/ts/bug.shtml.

If the documentation files are not in the help directory, you can:

  • Notify your system administrator about this problem, if the adminstator installed MATLAB.
  • Install MATLAB again to get the documentation files, if you installed MATLAB.
  • View the documentation at Documentation on the MathWorks Web site.
