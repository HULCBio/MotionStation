The Data Spectra Plot is opened by checking the corresponding check box in the ident window.

The plot shows the spectra of the output and input signals of selected data sets.

The method of computing/estimating the spectra is set using the Periodogram and Spectral analysis mutually exclusive checkmenus under the Options menu.

When Spectral analysis is chosen, the method and resolution parameter is determined by the current choice under popup menu Estimate > Spectral model in the ident window. Note that in this case only scalar resolution parameters are accepted for the SPAFDR case, and that the frequency range is always deternined from the plot windows's Options > Frequency range menu.

The periodogram is the absolute square of the Fourier transform of the data sequence.
