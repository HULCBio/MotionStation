To select experiments from a multiple experiment IDDATA set, drag and drop that set onto the Working Data. Then choose the item 'Select Experiments ...' in the Preprocessing pop-up menu.

The listbox in the dialog window that opens contains a list of the separate experiments present in the Working data set.

Select which experiment(s) you would like to study by clicking on the listbox. For multiple selctions and deselection, use ctrl+click. Enter a name for the selected data set in the edit box 'Data name'.

Press Insert to enter the new data set into the Data board.

Pressing the Revert button will remove your choices.
