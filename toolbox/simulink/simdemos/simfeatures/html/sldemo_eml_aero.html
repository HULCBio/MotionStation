Aircraft Position Estimation from Radar Measurements

Introduction

This model uses an Embedded MATLAB block to implement an extended Kalman filter. It is essentially the same as the long-time Simulink demo model aero_radmod that uses the aero_extkalman.m M-file, except the Embedded MATLAB version of the block in eml_aero_radar has enhanced I/O and supports code generation with Real-Time Workshop and production code generation with Real-Time Workshop Embedded Coder.

Demo Requirements

During this demonstration, Simulink and Real-Time Workshop generate code in a Simulink project directory created in the current working directory. If you do not want to (or if you cannot) generate files in this directory, you should change your working directory.
