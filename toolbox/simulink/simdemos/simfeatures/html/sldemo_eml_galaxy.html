Galaxy Simulation

Introduction

This model uses combined Embedded MATLAB blocks to simulate spiral arm formation of closely passing galaxies. It is inspired by the classic paper "Galactic Bridges and Tails", Toomre & Toomre 1972. Two disc shape galaxies are setup to almost collide. Once the galaxies are close enough, mutual gravitational forces give the two spiral arm effect.

Demo Requirements

During this demonstration, Simulink generates code in a Simulink project directory created in the current working directory. If you do not want to (or if you cannot) generate files in this directory, you should change your working directory.
