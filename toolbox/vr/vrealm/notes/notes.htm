2. Known Problems and Bugs.

 

  1. DEF names must be given to nodes before they can be used by the USE node. (default DEF names can not work).
  2. DEF names must be given to nodes before they can be used in the route editor. (default DEF names can not work.)
  3. When using the viewpoint node the new view position must be navigated to in the view window before adding the viewpoint. Then a description must be given.
  4. An undo performed on manipulator edit does not work until clicking in the node tree.
  5. Transparent properties are not always visible on materials and textures.
  6. The effects of collision under NavigationInfo node can not be seen.
  7. In Windows NT 3.51 ONLY, MovieTexture node will play audio but does not support video streaming. (Hear sound but see no picture.)
  8. In Windows NT 3.51 ONLY, multiple video and audio streams are not supported.
  9. In some isolated instances, support files are not saved (texture, audio, movie).
