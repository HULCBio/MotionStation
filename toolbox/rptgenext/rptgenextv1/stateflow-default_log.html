Description
Stateflow Default Report

This setup file produces a report which includes information about the
Stateflow information in the current model.  If there is no model open,
the report opens 'sf_car'.  If the current model does not include any
Stateflow blocks, the report will be  empty.  To specify the model
which the report will document, see the Model Loop component.
