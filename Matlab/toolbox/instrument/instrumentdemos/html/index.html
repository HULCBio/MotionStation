Instrument Control Toolbox Demos

The Instrument Control Toolbox provides MATLAB users with tools to allow them to communicate with instruments that support the GPIB (IEEE-488, HP-IB), VISA, serial port, TCPIP, and UDP interfaces.

A brief list of features includes 

  • Reading and writing of binary and ASCII data
  • Event-based communication 
  • Data and event recording
  • Tools that facilitate instrument control in an easy-to-use graphical environment 

The Toolbox currently supports GPIB boards provided by Agilent Technologies, Capital Equipment Corporation, CONTEC, ICS Electronics, IOTech, Keithley, Measurement Computing Corporation (ComputerBoards), and National Instruments. The Toolbox supports the VISA standard provided by Agilent Technologies, National Instruments, and Tektronix.
