Linearization GUI Introduction

This demonstration introduces you to the linearization graphical user interface (GUI) tools in Simulink Control Design. Opening the magballdemo model will:

  • Open the model magball demo which is a model of a magnetic ball levitation feedback control system.
  • The Control and Estimation Tools Manager launches and extracts a linear model of the closed loop system.

Once this demo is completed you can then explore the new features of the linearization tools in Simulink Control Design by reviewing the GUI overview.

Note: You can always launch the linearization GUI in the Control and Estimation Tools Manager by choosing Tools -> Control Design -> Linear Analysis from any Simulink model.
