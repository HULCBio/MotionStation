Hotkeys in the ident window:

You can speed up certain actions by pressing a key when the ident window is the current window. These keys are>

c creates a Correlation model.
s creates a Spectral model.
q performs Quick start.
d opens Import Iddata dialog.
m opens Import Model dialog.

Mouse buttons in the ident window:

A single click toggles selection for the data and model views.
Icons can by dragged-and-dropped by holding the mouse button down.
Double-clicking an icon opens an Info dialog.

Slightly faster responses can be obtained by different mouse buttons:
Use the left button to drag-and-drop.
Use the middle button (extend select) on a three-button mouse (same as shift+click) to toggle selection.
Right button (alt select), Cntrl+click and double-click any button all open the Info dialog.
