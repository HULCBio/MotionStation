Enter a row vector that contains the frequency values (in rad/s) to be used in the Frequency resp View.

Any one line MATLAB expression can be used to define this row vector.
Example 1: [1:100]*pi/100
Example 2: logspace(-3,-1,200)

When the frequency value option has been changed, all the curves in the Frequency resp View are redrawn for the new frequency values, except those curves that correspond to spectral analysis estimates. These will always be given for the frequency values that were chosen at the time the estimate was created.

The default value for the frequency values is 128 points linearly distributed up to and including the Nyquist frequency (not including frequency zero). The default value is obtained by entering the empty matrix.
