This dialog displays information about the data set and/or the model. A complete diary is displayed that gives a reproducible history of how the object has been created.

The name of the data set or model can be changed using the Data name: edit box.

Also the Diary and Notes can be edited, comments added, and so on.

The color of the icon - and all related curves - can be changed by entering either a color within quotes (e.g., 'y') or an RGB-triple (e.g., [1 0 0]) in the Color: edit box.

Present
Press Present to display added information to the MATLAB command line.
For models, a tailored presentation of the parameters is given. See PRESENT in the manual. For data sets, spectral and correlation analysis estimates, the data itself is displayed.
