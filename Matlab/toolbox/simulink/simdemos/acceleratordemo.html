Simulink Accelerator Demo

This demonstration introduces you to the Simulink Accelerator. Simulink Accelerator will not only speed up the execution of your model, but also allows you to switch back and forth between normal simulation and acceleration execution modes on the fly. This is especially useful when debugging your model with the Simulink graphical debugger. We can now step through a simple demonstration using the f14 model.

  • Open f14 model.
  • Set the stop time to 180 seconds to make things interesting.
  • Simulate the model and observe the output in the scopes.
  • Set the simulation mode to accelerated mode by clicking on Accelerator under Simulation menu.
  • The first time you accelerate the model, the accelerated model must be generated.
  • Simulate the model again, and observe the output in the scopes.
  • Notice the model being simulated faster.
Accelerator is particularly useful in concert with the debugger to accelerate to a given time, then debug at that time in normal mode.
