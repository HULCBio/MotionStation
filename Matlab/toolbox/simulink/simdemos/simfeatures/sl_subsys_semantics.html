Simulink Subsystem Semantics Demonstrations

This demonstration contains examples of different types of Simulink Subsystems and what sematics are used when simulating these Subsystems. Each demonstration provides a description of the model and the subtleties governing how it will be executed.

Examples are provided for the following Subsystem types.

  • Virtual and Nonvirtual Subsystems
  • Function-call Subsystems
  • Triggered Subsystems
  • Enabled Subsystems
  • Triggered and Enabled Subsystems
  • If/else Subsystems
  • While Subsystems
  • For Subsystems
