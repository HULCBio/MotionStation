Communication Toolbox MATLAB Functions and SIMULINK Blocks


This toolbox contains both SIMULINK function blocks and MATLAB functions. Most of the functionalities in the two parts overlap each other, although there are some exceptions. The functionality is divided into the following categories.

Sources and Sinks
Source Coding
Error-Control Coding
Modulation and Demodulation
Multiple Access
Transmitting and Receiving Filters
Channel
Synchronization
Utilities and Miscellaneous
Galois Field Computation

You can view a complete list of all SIMULINK blocks and a complete list of all MATLAB functions in this toolbox.
