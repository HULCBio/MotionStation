Real-Time Filter

This is an example of real-time filter built using the Signal Processing Blockset and Real-Time Windows Target. The unfiltered signal is acquired by the analog input, passed through a filter designed by the Signal Processing Blockset and finally sent to analog output. Both the unfiltered and filtered signals are shown in real-time using the Scope block.

How to run this demo:

  1. Open the Analog Input and Analog Output blocks and select your data acquisition board.
    If there's no board installed, install it using the Install new board button.
  2. Build the model using Tools -> Real-Time Workshop -> Build Model.
  3. Connect to the real-time code using Simulation -> Connect to target.
  4. Start the real-time execution using Simulation -> Start real-time code.

Note: To run this demo, you must have have a data acquisition board plugged into your computer.

Note: This demo requires the Signal Processing Blockset.
