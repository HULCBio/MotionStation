Real-Time Van der Pol Simulation

This demonstration shows a real-time version of the Simulink Van der Pol simulation demo. This model does not need any external signals, so it does not need any data acquisition hardware or driver. This makes the model ideal for first experiments with the Real-Time Windows Target, without need to worry about proper I/O hardware configuration.

How to run this demo:

  1. Build the model using Tools -> Real-Time Workshop -> Build Model.
  2. Connect to the real-time code using Simulation -> Connect to target.
  3. Start the real-time execution using Simulation -> Start real-time code.
