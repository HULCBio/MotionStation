Search Criteria

The Search Criteria panel allows you to specify properties that objects must have to satisfy your search request. The panel contains two subpanels. The Basic panel allows you to specify an object name and, optionally, a dialog parameter value that the object must satisfy. The Advanced panel allows you to specify these and other properties that the target of your search must satisfy.

Match case

If this option is checked, the object or parameter name must match the case as well as the text of the search expression.

Other Match Options

Select the drop-down button to display the following additional match options.

  • Match whole word

    Matches if the specified property value is the same as the search text except for case.

  • Contains word

    Match if the specified property value contains the search text.

  • Regular expression

    Match if the specified property value matches the regular expression represented by the search text.
