Basic

The Basic Criteria panel allows you to search for objects whose name and, optionally, dialog parameters match a specified text string.

Find what

Enter a name or parameter value in this field. The Finder searches for objects whose name or parameter values match the text that you enter in this field. To repeat a search, select the dropdown button adjacent to the Find what field and select the previous search text from the resulting dropdown list.

Search block dialog parameters

Check this option if you want the Finder to search for objects whose dialog parameter values match the text entered in the Find what field.
